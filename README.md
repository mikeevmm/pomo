# POMO

A command-line based pomodoro timer.

![Pomo demo](demo.gif)

Run `pomo`, list your tasks and start working.
`pomo` will count your pomodoros and breaks for you.

## Quick Start

```bash
git clone https://github.com/mikeevmm/pomo
cd pomo 
bash install.sh
pomo --help
```
